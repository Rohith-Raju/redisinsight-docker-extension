## Getting Started

## Clone the repository


```shell
 git clone https://github.com/collabnix/redisinsight-docker-extension
```

## Build the Extension

```shell
 make build-extension
```

## Install the Extension

```shell
 docker extension install <name-of-extension>
 ```
